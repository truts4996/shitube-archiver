# shitube-archiver
a very simple python script to download youtube videos, playlists and channels. meant for archiving deleted/terminated youtube channels via wayback machine snapshots.

# requirements
1. Python
2. yt-dlp
3. ffmpeg (optional)

# installation instructions
1. download Python from the official website or use Ninite/Chocolatey/Scoop/winget
2. download yt-dlp from the official GitHub repository and place it inside shitube-archiver's folder, download via pip with ```pip install yt-dlp``` or use Chocolatey/Scoop/winget
3. download ffmpeg from the official website and place it inside shitube-archiver's folder or use Chocolatey/Scoop/winget

# how to use
1. find a channel/video/playlist on wayback machine
2. copy the ink
3. open the script
4. paste it
5. press enter
